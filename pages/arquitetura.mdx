# Configuração de Ambiente

### Introdução à Arquitetura do Ecossistema

Antes de prosseguirmos com a configuração do ambiente, é fundamental compreender a estrutura do nosso ecossistema. Atualmente, o ecossistema é segmentado em várias aplicações que se integram para fornecer os serviços do Launcher 360 e da plataforma online myAudaces.

#### Repositórios Relevantes

Temos quatro repositórios principais que você deve conhecer:

- **Launcher360**: Este repositório contém o código relacionado ao Launcher 360, desenvolvido em Electron. 

- **Launcher360 Marketing Manager Front**:  Este repositório refere-se à plataforma onde é possível gerenciar os banners exibidos no Launcher360.

- **MyAudaces-website**: Este é o repositório relacionado à plataforma web MyAudaces.

- **Api Launcher360**: Este repositório backend fornece APIs a serem consumidas nos três repositórios mencionados anteriormente, é o ponto central do ecossistema.


Agora que temos uma visão geral da arquitetura, vamos passar para a configuração do ambiente em sistemas Windows.
